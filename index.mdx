## Packages

| Package | Description |
|---------|-------------|
| `@chalkysticks/sdk` | Main SDK aggregator |
| `@chalkysticks/sdk-core` | Foundation: models, collections, events |
| `@chalkysticks/sdk-pad` | Pad/diagram functionality |
| `@chalkysticks/sdk-game` | Game physics and logic |
| `@chalkysticks/sdk-venues` | Venue data and search |
| `@chalkysticks/vue2-components` | Reusable Vue components |
| `@chalkysticks/vue2-trainer` | Pool simulator |
| `@chalkysticks/vue2-pad` | Diagram tool |
