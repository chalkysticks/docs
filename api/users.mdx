---

## List Users

```bash
GET /v3/users
GET /v3/users?q=matt
GET /v3/users?limit=25
```

## Get User

```bash
GET /v3/users/{id}
```

**Response:**
```json
{
  "data": {
    "id": 1,
    "name": "Matt Kenefick",
    "avatar": "https://...",
    "created_at": "2025-01-01T00:00:00Z"
  }
}
```

## Update Current User

```bash
PUT /v3/users/me
Authorization: Bearer YOUR_TOKEN
{
  "name": "New Name",
  "bio": "Updated bio"
}
```

## User Media

### List Media

```bash
GET /v3/users/{id}/media
```

### Upload Media

```bash
POST /v3/users/{id}/media
Authorization: Bearer YOUR_TOKEN
Content-Type: multipart/form-data

file: [binary]
```

### Confirm Upload (Presigned)

```bash
POST /v3/users/{id}/media/confirm-upload
Authorization: Bearer YOUR_TOKEN
{
  "key": "uploads/abc123.jpg"
}
```

### Update Media Order

```bash
PATCH /v3/users/{id}/media/{media_id}/order
Authorization: Bearer YOUR_TOKEN
{
  "order": 2
}
```

## Wallet

### Get Wallet

```bash
GET /v3/users/{id}/wallet
Authorization: Bearer YOUR_TOKEN
```

**Response:**
```json
{
  "data": {
    "balance": 150,
    "lifetime_earned": 500
  }
}
```

### Collect Points

```bash
POST /v3/users/{id}/wallet/collect
Authorization: Bearer YOUR_TOKEN
{
  "amount": 10,
  "source": "daily_login"
}
```

### Spend Points

```bash
POST /v3/users/{id}/wallet/spend
Authorization: Bearer YOUR_TOKEN
{
  "amount": 50,
  "product_id": 1
}
```
