## Beacons

Location broadcasts from players.

### List Beacons

```bash
GET /v3/beacons
GET /v3/beacons?lat=40.7128&lon=-74.0060&distance=10
```

### Create Beacon

```bash
POST /v3/beacons
Authorization: Bearer YOUR_TOKEN
{
  "latitude": 40.7128,
  "longitude": -74.0060,
  "message": "Looking for a game!"
}
```

### Delete Beacon

```bash
DELETE /v3/beacons/{id}
Authorization: Bearer YOUR_TOKEN
```
