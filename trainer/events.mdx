## Overview

All events are dispatched via `ChalkySticks.Core.Event.Bus` and defined as enums in `src/App/Event/`. Import them as:

```typescript
import * as Event from '../Event';

// Listen
ChalkySticks.Core.Event.Bus.on(Event.Shot.Complete, handler);

// Dispatch
ChalkySticks.Core.Event.Bus.dispatch(Event.GameControl.Shoot);
```

There are 30 event namespaces covering every aspect of the game engine.

---
