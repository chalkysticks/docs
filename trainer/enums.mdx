## Overview

All enums are defined in `src/App/Enum/` and imported as a namespace:

```typescript
import * as Enum from '../Enum';

// Usage
const gameType = Enum.RackStrategy.EightBall;
const mode = Enum.MatchMode.VersusCpu;
```

There are 25 enums covering game types, states, strategies, and configuration options.
